基本的な使い方

  • 名前とパスワードを入力して投稿できます。
  • パスワードはあなたのIDを決めるものです。毎回同じパスワードを使えば同じIDになります。
  • IDは @xxxxxxx の形式（パスワードのハッシュ値）で自動生成されます。
  • 投稿は1秒に1回まで。
  • >>番号 で特定の投稿にアンカーを付けられます。
  • URLは自動でリンクになります。画像URLは自動でサムネイル表示されます（タップで拡大）。
  • 絵文字ボタン（😊）から絵文字を挿入できます。

権限一覧

運営
IDが赤・太字で表示。すべてのコマンドが使用可能。
サミット
IDがダークシアン。運営の代理でほぼすべてのコマンドが使用可能。
マネージャー
IDが赤色。ある程度のコマンドが使用可能。
スピーカー
IDが薄橙（オレンジ）。コマンドは使えません。
青ID
IDが青色。初期権限。コマンドは使えません。各種規制がかかります。

投稿削除コマンド

コマンド説明権限
/del 番号 指定した番号の投稿を削除。複数指定可（スペース区切り）。
例: /del 12 26		 マネージャー以上
/destroy 文字列 指定した文字列またはIDを含む投稿を一括削除。複数指定可。
色指定も可: blue 青ID・darkorange スピーカー・red マネージャー/運営・darkcyan サミット		 マネージャー以上
/clear すべての投稿を削除（固定投稿no=0〜2は残る）。投稿番号もリセット。 サミット以上

投稿規制コマンド

コマンド説明権限
/NG 言葉 指定した言葉を青IDの投稿禁止ワードに登録。複数指定可。 マネージャー以上
/OK 言葉 NGワードを解除。複数指定可。 マネージャー以上
/prevent 解除されるまで青IDが雑談に投稿できなくする。
話題欄に「雑談での青IDの発言制限中」と表示。		 サミット以上
/permit /prevent を解除。 マネージャー以上
/restrict banされた青IDが全サーバーで投稿できなくなる。
話題欄に「青ID一部規制中」と表示。		 サミット以上
/stop 3分間、青IDが全サーバーで投稿できなくなる。
話題欄に禁止終了時刻を表示。		 サミット以上
/prohibit 時間 指定した時間、青IDが全サーバーで投稿できなくなる。
時間指定: 3h（3時間）・30m（30分）・2h30m（2時間30分）
例: /prohibit 1h30m
話題欄に禁止終了時刻を表示。		 運営のみ
/release /restrict・/stop・/prohibit をすべて解除。 マネージャー以上

コマンド説明権限
/speaker ID 指定IDをスピーカーに昇格。複数指定可。
例: /speaker yukibbs		 マネージャー以上
/manager ID 指定IDをマネージャーに昇格。複数指定可。 サミット以上
/summit ID 指定IDをサミットに昇格。複数指定可。 運営のみ

権限降格コマンド

コマンド説明権限
/disspeaker ID スピーカーを青IDに降格。複数指定可。 マネージャー以上
/dismanager ID マネージャーをスピーカーに降格。複数指定可。 サミット以上

アカウント操作コマンド

コマンド説明権限
/kill ID 指定IDを使用不可にする（再起動か/reviveまで）。複数指定可。 サミット以上
/ban ID
/ban 投稿番号		 指定IDまたは投稿番号のIDをban。/restrict作用中に全投稿を禁止。複数指定可。 サミット以上
/revive すべてのkill・banを解除。 マネージャー以上

一括実行について

以下のコマンドはスペース区切りで複数の対象を一括指定できます：

  • /del 12 26 30 — 複数番号を一度に削除
  • /destroy yukibbs blue — 複数条件を一度に削除
  • /NG 言葉1 言葉2 — 複数NGワードを一度に登録
  • /OK 言葉1 言葉2 — 複数NGワードを一度に解除
  • /speaker ID1 ID2 — 複数IDを一度に昇格
  • /kill ID1 ID2 / /ban ID1 ID2 — 複数IDを一度に操作
