エンドポイント一覧

すべて同一サーバー（https://yuyuyu-made-bbs-server.onrender.com）で提供しています。

メソッドパス説明
GET /api 雑談の投稿・topic・規制状態を取得
POST /api 雑談に投稿（またはコマンド実行）
GET /api/battle バトルスタジアムの投稿・topic・規制状態を取得
POST /api/battle バトルスタジアムに投稿（またはコマンド実行）
GET /health サーバーの死活確認

データ取得（GET）

GET https://yuyuyu-made-bbs-server.onrender.com/api
GET https://yuyuyu-made-bbs-server.onrender.com/api/battle

namestring投稿者名
contentstring投稿内容
idstringハッシュID（@xxxxxxx形式）
timestring投稿日時（JST）
deletedboolean削除済みかどうか

投稿送信（POST）

POSTリクエストのみ受け付けます。送信は1秒に1回まで。

方法1: クエリパラメーター

POST https://yuyuyu-made-bbs-server.onrender.com/api?name=名前&pass=パスワード&content=内容

方法2: JSONボディ

POST https://yuyuyu-made-bbs-server.onrender.com/api
Content-Type: application/json

{
  "name": "名前",
  "pass": "パスワード",
  "content": "内容"
}

リクエストフィールド

フィールド必須説明
name✓投稿者名（最大50文字）
pass✓パスワード（IDの生成に使用。SHA-256ハッシュの先頭7文字がIDになる）
content✓投稿内容（最大1000文字）。コマンドもここに記述。

レスポンス例（成功）

{
  "message": "投稿成功",
  "post": { "no": 6, "name": "名前", ... }
}

エラーレスポンス例

{
  "error": "投稿が禁止されています"
}

IDの仕組み

パスワードをSHA-256でハッシュ化し、先頭7文字に@を付けたものがIDになります。

ID = "@" + sha256(パスワード).slice(0, 7)

例えばパスワードを 7 にすると ID は @7902699 になります。

同じパスワードを使い続けることで同じIDを維持できます。
