掲示板に戻る

データ取得

https://yuyuyu-made-bbs.onrender.com/apiで投稿とtopicが取得できます。(json形式)

投稿送信

(POSTリクエストでないと送信できません。)

送信できるのは1秒1回です。

方法1: クエリパラメーター

https://yuyuyu-made-bbs-server.onrender.com/api?name=[名前]&pass=[パスワード]&content=[内容]

にPOSTリクエストで送信

方法2: JSON

https://yuyuyu-made-bbs-server.onrender.com/api

にPOSTリクエストで以下のJSONを送信：

{
  "name": "名前",
  "pass": "パスワード", 
  "content": "内容"
}

Content-Type: application/json を設定してください。

投稿データ形式

投稿には自動で番号が付与されます：

{
  "no": 3,
  "name": "ユーザー名",
  "content": "投稿内容",
  "id": "@abc1234",
  "time": "2025/8/31 15:30:00"
}
